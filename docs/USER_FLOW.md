# Fluxo de usuário

## Tela inicial

O app abre com três opções:

- Bluetooth direto.
- Wi-Fi / Hotspot.
- Modo demonstração.

Nenhum aviso fixo sobre instalação aparece na tela inicial.

## Wi-Fi / Hotspot

Ao escolher Wi-Fi / Hotspot, o app mostra a tela do Companion com link de download, ações para copiar/compartilhar/abrir o link e o botão "Já abri o Companion".

## Controle remoto

A tela de controle mantém o celular ligado durante a apresentação, mostra status, timer e botões para avançar, voltar, iniciar, encerrar, tela preta e tela branca.
