# Limitações do Bluetooth direto

O modo Bluetooth direto depende do suporte do aparelho Android para atuar como dispositivo HID/presenter.

Em muitos aparelhos esse recurso não está disponível para apps comuns ou exige suporte específico do fabricante. Quando o suporte não existir, o app deve mostrar uma mensagem amigável e oferecer o modo Wi-Fi / Hotspot com Companion.

Mensagem recomendada:

```text
Este aparelho não oferece suporte ao modo Bluetooth direto.
Você ainda pode usar o modo Wi-Fi/Hotspot com Companion.
```
