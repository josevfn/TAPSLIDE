# Android App

App Android do Slide Remote, criado com Kotlin, Jetpack Compose e Material 3.

## Build

Windows PowerShell:

```powershell
.\gradlew.bat assembleDebug
```

Linux/macOS:

```bash
./gradlew assembleDebug
```

## Fase 1

- Tela inicial limpa.
- Tela de download do Companion.
- Ações para copiar, compartilhar e abrir link.
- Tela de pareamento Wi-Fi inicial.
- Modo demonstração.
- Tela de controle remoto com timer e tela sempre ligada.
